# weather-app

It has 2 services
  api-service : which exposes endpoints to cities 
  weather-service: service which reads weather data from weather api
  
  
  To run api-service
   `cd api-service`
   `npm install`
   `npm run start`
   
   The server will be running in port 3000
   
   To run weather-service
   `cd weather-service`
   `npm install`
   `npm run start`
